Tic-Tac-Toe with DxLib
C++とＤＸライブラリを使用して作成された、デスクトップ向け三目並べ（まるばつゲーム）です。
直感的なマウス操作と、正確な勝利判定アルゴリズムを備えています。

🎮 ゲーム概要
対戦形式: ローカル2人対戦（○と×を交互に配置）。

操作方法: マウス左クリックでマスを選択。

ルール: 縦・横・斜めのいずれかに3つ同じ記号を並べると勝利。すべてのマスが埋まると引き分けです。

✨ 特徴
正確な入力検知: 毎フレームの入力状態を比較し、マウスボタンが「押された瞬間」のみを判定することで、意図しない連続入力を防止しています。

動的スケーリング: CELL_SIZE や OFFSET の定数定義により、画面中央への正確なグリッド描画とレイアウト計算を行っています。

勝利判定ロジック:

3×3の二次元配列をループと条件分岐でスキャンし、8パターンの勝利ラインと引き分け状態を瞬時に判定します。

🛠 技術仕様
言語: C++

グラフィックスライブラリ: ＤＸライブラリ (DxLib)

主な使用関数:

DrawLine, DrawCircle: 盤面および記号の描画。

GetMouseInput, GetMousePoint: マウス座標とクリック状態の取得。

CheckHitKey(KEY_INPUT_ESCAPE): 安全なアプリケーション終了処理。

🚀 実行方法
環境構築: Visual Studio 等のIDEにＤＸライブラリを導入します。

ソースの追加: main.cpp としてこのコードを追加します。

ビルド・実行: ウィンドウモードで起動し、マウスで操作します。

📂 ファイル構成
main.cpp: ゲームループ、入力処理、描画処理、勝利判定ロジックをすべて含むメインソースコード。
